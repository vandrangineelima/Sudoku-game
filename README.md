## 📌 Project Description
The **Sudoku Game** is an interactive browser-based puzzle game developed using **HTML, CSS, and JavaScript**.  
The objective of the game is to correctly fill the Sudoku grid according to the rules of Sudoku.

This project provides a clean and user-friendly interface with different difficulty levels, making it both educational and fun.

---

## 🛠️ Technologies Used
- **HTML**
- **CSS**
- **JavaScript**

---

## ✨ Features
- 🧩 **Sudoku Puzzle Gameplay**
  - Fill the grid correctly according to Sudoku rules

- 🎯 **Multiple Difficulty Levels**
  - Easy
  - Medium
  - Difficult

- ✅ **Check Solution Button**
  - Verifies whether the entered solution is correct

- 🔄 **Reset Button**
  - Resets the puzzle for a fresh start

- 🌙 **Dark Mode**
  - Toggle between light and dark theme

- 💬 **Game Messages**
  - Displays feedback messages to guide the player
